## The @since Tag ##

The `@since` tag indicates that a method or property of another class has been added at a given version number.

### Syntax ###

```
@since versionDescription
```

  * versionDescription - Required: a description of the version.

### Example ###

```
/** @since version 1.0.1 */
function Remote() {
}
```
